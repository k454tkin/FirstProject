Тег "tfoot"

Описание:

Элемент <tfoot> предназначен для хранения одной или нескольких строк, которые представлены внизу таблицы. Хотя тег <tfoot> в исходном коде должен быть определен до тега <tbody>, браузеры отображают его в самом низу таблицы. В пределах таблицы разрешается использовать только один элемент <tfoot>.
Синтаксис
<table>
<thead>...</thead>
<tfoot>
<tr>
<td>Текст</td>
</tr>
</tfoot>
<tbody>
...</tbody>
</table>

Атрибуты:

align Определяет выравнивание текста в строке.
bgcolor Цвет фона ячеек, которые расположены внутри контейнера <tfoot>.
char Выравнивание содержимого ячеек относительно заданного символа.
charoff Смещение содержимого ячеек относительно указанного символа.
valign Выравнивание содержимого ячеек по вертикали.
