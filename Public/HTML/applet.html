Тег "applet"

Описание:

Тег <applet> предназначен для вставки на страницу апплетов — небольших программ, написанных на языке Java.
Этот тег является устаревшим, взамен необходимо использовать более гибкий тег <object>. Между открывающим и закрывающим тегом можно добавить текст, который будет отображаться в браузере, если он не поддерживает апплеты. В противном случае текст не выводится.

Атрибуты:

  1. align
    Задает выравнивание апплета относительно близлежащих элементов и текста.
    Синтаксис - <applet align="bottom | left | middle | right | top"></applet>
    Примеры значений

  2. alt
    Атрибут alt устанавливает альтернативный текст для апплетов.
    Такой текст позволяет получить информацию о программе во время ее загрузки.
    Обязательно брать в двойные или одинарные кавычки.
    Синтаксис - <applet code="URL" alt="текст"></applet>
    Ни один из браузеров альтернативный текст не отображает каким-либо образом.

  3. archive
    Позволяет указать архивный файл в формате ZIP или JAR (Java-архив), в котором располагается апплет.
    Синтаксис - <applet code="URL" archive="URL файла с архивом"></applet>
    В качестве значения принимается полный или относительный путь к архивному файлу.

  4. code
    Указывает имя файла с Java-классом (оно имеет расширение .class). Это имя может комбинироваться со значением атрибута codebase, который задает путь к папке, где располагается файл.
    Синтаксис - <applet code="URL"></applet>
    Имя файла с расширением .class. Если расширение не указано (например: code="clock"), то браузер добавит расширение автоматически.

  5. codebase
    Используется для установки базового адреса, относительно которого будет задаваться значение атрибута code. Если значение codebase не указано, то в качестве адреса принимается адрес текущего документа.
    Синтаксис - <applet code="URL" codebase="URL"></applet>
    Абсолютный или относительный путь к папке, где хранятся файлы с Java-классами.

  6. height
    Задаёт высоту апплета. Допускается использовать значения в пикселях или процентах. Если установлена процентная запись, то размеры апплета вычисляются относительно размера родительского элемента — контейнера, где находится тег .
    Синтаксис - <applet code="URL" height="значение"></applet>
    Браузеры по разному задают исходную высоту апплета:
    Internet Explorer — 150 пикселов,
    Firefox — 198 пикселов,
    Opera — 150 пикселов,
    Chrome — 50 пикселов.

  7. hspace
    Задает горизонтальный отступ вокруг апплета.
    Синтаксис - <applet code="URL" hspace="отступ по горизонтали"></applet>
    Значение по умолчанию 0
    Валидация - использование этого атрибута осуждается спецификацией HTML, взамен рекомендуется использовать стили.

  8. vspace
    Задает вертикальный отступ вокруг апплета.
    Синтаксис - <applet code="URL" vspace="отступ по вертикали"></applet>
    Значение по умолчанию 0
    Валидация - использование этого атрибута осуждается спецификацией HTML, взамен рекомендуется использовать стили.

  9. width
    Задаёт ширину апплета. Допускается использовать значения в пикселях или процентах. Если установлена процентная запись, то размеры апплета вычисляются относительно размера родительского элемента — контейнера, где находится тег . В случае отсутствия родительского контейнера, в его качестве выступает окно браузера. Иными словами, width="100%" означает, что апплет будет растянут на всю ширину веб-страницы.
    Синтаксис - <applet code="URL" width="значение"></applet>
    Значения - любое целое положительное число в пикселях или процентах.
    Браузеры по разному задают исходную ширину апплета:
    Internet Explorer — 150 пикселов,
    Firefox — 198 пикселов,
    Opera — 150 пикселов,
    Chrome — 50 пикселов.
