Тег "bgsound"

Описание:

Тег <bgsound> определяет музыкальный файл, который будет проигрываться на веб-странице при ее открытии. Громкость, продолжительность звучания музыки и другие характеристики определяются с помощью атрибутов тега, а также могут управляться через скрипты. Этот тег должен размещаться только в контейнере <head>.
Тег <bgsound> не входит в спецификацию HTML и при его использовании код не пройдет валидацию. К тому же пользователи, как правило, отрицательно относятся к музыке, которая играет на сайте, поэтому используйте эту возможность с осторожностью и по необходимости.
Синтаксис:
<head>
<bgsound атрибуты>
</head>

Атрибуты:

  1. balance
    Регулирует громкость звучания в левой и правой колонках. Этот атрибут не может быть изменен динамически с помощью скриптов.
    Синтаксис - <bgsound balance="число">
    Целое число от -10000 до 10000. Значение ноль служит для баланса громкости, отрицательные числа увеличивают громкость в левой колонке, а положительные — в правой.

  2. loop
    Устанавливает, сколько раз проигрывать музыкальный файл. Если значение отсутствует, то музыка проигрывается только один раз.
    Синтаксис - <bgsound loop="число">
    Значение 0 проигрывает файл один раз, -1 проигрывает музыку до тех пор, пока текущая веб-страница открыта. Любое целое положительное число заставляет браузер проигрывать музыкальный файл указанное число раз. По умолчанию значение -1.

  3. src
    Путь к музыкальному файлу, который будет проигрываться на веб-странице. Список допустимых типов файлов зависит от браузера и его настроек, но популярные форматы, например, MIDI поддерживается браузером без установки дополнительных плагинов.
    Синтаксис - <bgsound src="URL">
    В качестве значения принимается полный или относительный путь к файлу.

  4. volume
    Задает громкость звучания музыки на веб-странице. По умолчанию уровень звука устанавливается максимальным и зависит от текущих настроек оборудования и операционной системы. Поэтому громкость можно только уменьшать, присваивая атрибуту volume отрицательное значение. Чем это число больше, тем тише звучит музыка на веб-странице.
    Синтаксис - <bgsound volume="число">
    Целое число от -10000 до 0. Ноль соответствует максимальной громкости звучания. По умолчанию 0.
